# d3.js

  • Observable: 최근 (2018년 1월) 릴리즈된 d3용 노트북 서비스 👍
  • D3 blocks
  • D3 Discovery: d3 관련 페이지를 잘 모아놨다.
  • The Big List of D3.js Examples: 2000여개의 링크를 모아두었다. 깨진 링크가 많지만, 그래도 훑어보기에 유용하다.
  • d3.js 책 속 추천목록
    • DataWrapper: 최신브라우저에선 인터렉티브, 오래된 브라우저엔 정적 이미지로 표시한다. 저널리스트를 위해서 설계된 서비스.
    • dc.js: crossfilter를 이용한 개량된 버전인데, 물건이다.
    • billboard.js: naver 개발자가 만든 c3.js 개량버전. 2017년까지는 활발하게 개발했었다. D3.js v4를 지원한다.
  • Vega
  • Vega-lite: d3.js의 원산지였던 IDL의 라이브러리
    • Vega-lite 예제: 월별 강수량: 아래 코드를 이용하여 테스트할 수 있다.
{
  "$schema": "https://vega.github.io/schema/vega-lite/v3.json",
  "data": {"url": "https://api.taegon.kr/stations/108/?sy=2016&ey=2016"},
  "mark": "bar",
  "encoding": {
    "x": {"timeUnit": "month", "field": "Date", "type": "ordinal"},
    "y": {"aggregate": "sum", "field": "rainfall", "type": "quantitative"}
  }
}
